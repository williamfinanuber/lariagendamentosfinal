# **App Name**: Edi Nails Scheduler

## Core Features:

- Availability Calendar: Display available appointment dates and times based on admin's setup.
- Procedure Selection: Allow customers to select a specific nail procedure and view its details and pricing.
- Appointment Booking: Enable customers to book appointments within the available time slots.
- Procedure Management: Provide admin panel for uploading images, descriptions, and pricing details for each nail procedure.
- Availability Setting: Allow the admin to specify the dates and times that she is available for bookings.
- Notifications: Enable notifications (e.g., SMS, email) for new bookings and appointment reminders. AI may use tool to customize notification texts.
- Customer Profile: Basic customer data capture.

## Style Guidelines:

- Primary color: Deep, saturated purple (#673AB7), associated with creativity and beauty.
- Background color: Very light purple (#F3E5F5), offering a soft, clean backdrop.
- Accent color: Pink (#E91E63), which is analogous to the primary, adds vibrancy as a highlight for calls to action and interactive elements.
- Font: 'Poppins', a geometric sans-serif for a contemporary and fashionable look, suitable for UI elements and short amounts of text.
- Use elegant line icons representing nail art and beauty treatments.
- Clean, grid-based layout for procedure and availability display.
- Subtle transitions for navigation and booking confirmations.
